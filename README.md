# Is It Me? 🦝

> A community bug tracker for working in tech.  
> File your issues. We won't fix them either.  
> But at least you'll know it's not just you.

---

## What is this?

This is a GitHub repository where the issues ARE the community.

If you work in tech and you've ever thought:

- "Is this actually fucked up or is it just me?"
- "Did that really just happen in a meeting?"
- "Is this man lying to me?"
- "Am I the problem or is the problem the problem?"
- "I can't say this anywhere else but I need to say it"

You're in the right place. File an issue. We'll triage it together.

---

## How to file an issue

1. Hit **New Issue**
2. Give it a title. Be specific. "My manager" is not a title. "My manager told me to take it as a compliment" is a title.
3. Describe what happened. As much or as little as you want.
4. Pick your labels (see below)
5. Submit. Watch the community tell you it's not just you.

---

## Labels

| Label | Meaning |
|---|---|
| `not-just-you` | Confirmed. This is a them problem. |
| `classic-bro-move` | We've seen this before. Many times. |
| `working-as-intended (unfortunately)` | The system is functioning exactly as designed. That's the problem. |
| `known-issue` | Everyone knows. Nobody fixes it. |
| `wont-fix` | The cruelest status. Also the most accurate. |
| `is-it-me` | Genuinely unclear. Community input needed. |
| `it-was-not-you` | Closed with verdict. |
| `help-needed` | You need actual advice, not just validation. Both are valid. |
| `omg` | No further notes. |

## Rules

**No bros.** This isn't about gender. It's about energy. You know if you're a bro. Bro energy includes: explaining things to people who didn't ask, making it about you, and telling people to "just be more confident."

**No fixing unsolicited.** Sometimes people need to be heard. Read the labels. If someone hasn't tagged `help-needed` they probably just need a `not-just-you` and a cup of tea.

**No naming and shaming.** File the behaviour, not the LinkedIn profile.

**No minimising.** "It could be worse" is not a comment. It goes in the bin.

**Raccoons only.** We are feral, we are unglamorous, we make do, and we look out for each other. Welcome to the dumpster.
